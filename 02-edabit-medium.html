Example 02: Perimeters with a Catch

Question: Write a function that takes a number and returns the perimeter of either a circle or a square. The input will be in the form (letter l, number num) where the letter will be either "s" for square, or "c" for circle, and the number will be the side of the square or the radius of the circle.

The catch is you can only use arithmetic or comparison operators, which means: No if... else statements. No objects. No arrays. No formatting methods, etc.

The goal is to write a short, branch-free piece of code.

  • perimeter("s", 7) ➞ 28 - Haha
  • perimeter("c", 4) ➞ 25.12 -
  • perimeter("c", 9) ➞ 56.52 -
